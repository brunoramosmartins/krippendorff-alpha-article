# About

This project supports a portfolio-grade technical article on inter-annotator agreement: how naive **observed agreement** mixes signal and chance, how **Kappa**-style corrections help yet fail under imbalance and missing data, and how **Krippendorff's Alpha** generalises across raters and scales by modelling disagreement.
